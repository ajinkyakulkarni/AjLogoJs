# AjLogoJs

Logo interpreter written in Javascript.

## Installation

Via npm on Node:

```
npm install ajlogo
```

## Usage

Reference in your program:

```js
var ajlogo = require('ajlogo');
```

In your browser:

```html
<script src="ajlogo.js" type="text/javascript"></script>
```
